Colophon

This website runs on Google App Engine. It is written in Python on the back end and HTML, CSS and JavaScript for the front end. It uses the Google Datastore for most of the database stuff, except the images which are stored and manipulated by the Google Blobstore.

Julia Belling did the lovely LBi-esque design and Paul Carvill did the code. Photos are the models' own and are not actual size.

All the code is in GitHub here http://ww.github.com/paulcarvill so please check it out and help us improve the site.
